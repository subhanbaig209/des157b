Add a short description of your function, what it does and where you got it. You can pull it from some other project you have, or you can create a function from scratch that does something simple and straightforward. Don't just use one of Bill and Glenda's examples though.

Named Function

A named function that multiplies two numbers together.

            function multiplyNumbers(a, b) {
                return a * b;
            }

Function Expression

            const multiplyNumbers = function(a, b) {
                return a * b;
            };

Arrow Function

            const multiplyNumbers = (a, b) => {
                return a * b;
            };
